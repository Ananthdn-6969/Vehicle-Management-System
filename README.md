# Vehicle Management System

A Vehicle Management System developed using Flask and MySQL.

## Features

- Vehicle Management
- Owner Management
- Vehicle-Owner Mapping
- Insurance Records
- Service History
- Fuel Log

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- Bootstrap

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a MySQL database named `DBMS`

4. Run the SQL script (`database.sql`)

5. Update the MySQL credentials in `App.py`

6. Run the project

```bash
python App.py
```
